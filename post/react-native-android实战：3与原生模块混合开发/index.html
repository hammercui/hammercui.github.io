1 回调

java层需要把结果传递给js，答案就是回调，最典型的一个场景就是javascript层调用java层的网络请求方法，java层拿到网络数据后需要将结果返回给javascript层

为了测试我们在已建好的模块新增方法,invoke回调的参数是无类型，无个数限制的
